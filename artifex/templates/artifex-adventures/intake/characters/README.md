# Characters Intake

Drop hero art, NPC art, villains, enemies, portraits, turnarounds, sprite sheets, animation frames, and character references here.

Examples:

```text
tommy_artifex_turnaround.png
tommy_artifex_walk_sheet_draft.png
evelyn_hastings_portrait.png
tracker_mobster_reference.png
```

These files are staging files. Final approved character assets should later be copied into the correct `assets/characters/` folder and indexed by the Asset Library.
