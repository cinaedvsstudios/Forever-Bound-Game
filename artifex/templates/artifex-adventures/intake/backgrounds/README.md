# Backgrounds Intake

Drop full scene backgrounds, title/ending backgrounds, train interiors, station platforms, corridors, scenery loops, window tiles, parallax plates, and environmental plates here.

Examples:

```text

window_view_day_tile.jpg
```

These files are staging files. Final approved backgrounds should later be copied into the correct `assets/backgrounds/` folder and indexed by the Asset Library.
