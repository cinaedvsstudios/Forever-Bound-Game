# Dialogue and Sound Effects Intake

Drop spoken dialogue, narration, train ambience, footsteps, door clicks, drawer sounds, object pickup sounds, UI beeps, warning sounds, and other non-musical audio here.

Examples:

```text
dialogue_evelyn_ransom_message_draft.wav
sfx_train_wheels_loop_draft.wav
sfx_cabinet_drawer_open_draft.wav
sfx_item_pickup_draft.wav
```

These files are staging files. Final approved audio should later be copied into the correct `assets/audio/dialogue/`, `assets/audio/ambience/`, or `assets/audio/sfx/` folder and indexed by the Asset Library.
