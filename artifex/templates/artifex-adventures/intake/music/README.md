# Music Intake

Drop musical assets here: title themes, hero themes, location music, tension cues, chase music, victory music, menu music, and musical stingers.

Examples:

```text
music_artifex_adventures_hero_theme_draft.mp3
music_compartment_4b_ambience_theme_draft.ogg
music_vienna_victory_draft.mp3
```

These files are staging files. Final approved music should later be copied into the correct `assets/audio/music/` folder and indexed by the Asset Library.
