# Artifex Adventures Intake Folder

This is the simple creator-facing drop zone for unfinished or newly supplied Artifex Adventures assets.

Put files here first when making or collecting artwork, music, icons, character art, object art, dialogue, or sound effects. Later, Artifex can import them, rename them safely, copy them into the correct final `assets/` folder, create Asset Library records, and update game references to the clean final paths.

Use these buckets only:

```text
backgrounds/
characters/
objects/
icons-ui/
music/
dialogue-sfx/
```

Do not make permanent scene, object, quest, or runtime references directly to files inside `intake/`. Intake files are staging files, not final game assets.
