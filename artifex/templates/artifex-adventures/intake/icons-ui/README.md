# Icons and UI Intake

Drop inventory icons, action icons, interact prompts, menu icons, HUD pieces, buttons, panels, UI frames, and status symbols here.

Examples:

```text
icon_interact.png
icon_inventory_postcards.png
button_start_game.png
panel_dialogue_frame.png
```

These files are staging files. Final approved UI assets should later be copied into the correct `assets/ui/` folder and indexed by the Asset Library.
