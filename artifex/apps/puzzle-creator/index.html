Build creates or parses the maze. Display controls the 3D presentation. Logic controls the game-readable quest metadata. Materials changes the theme. Paint lets you edit wall cells directly.

Controls: Diorama mode rotates with mouse drag. Walk Test uses WASD or arrow keys. Paint and Draw work on either the 2D matrix or the 3D wall blocks.
