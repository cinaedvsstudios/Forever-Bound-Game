# Pattern Lock module

This folder contains a standalone Pattern Lock puzzle module.

The uploaded recovery pack only contained downloader placeholders for the historical Pattern Lock files, so this folder includes a clean offline standalone rebuild with the same intended behaviour:
- selectable cube, pyramid, diamond and sphere templates
- rotatable surface point puzzle
- emoji/sign tray
- erase/clear/show-correct-example tools
- whole-pattern validation without revealing each wrong point

Open `index.html` directly.
