Artifex FX Editor refactor pass 1

What changed:
- Split the original single-file index.html into a small HTML shell plus external CSS and JavaScript.
- Moved custom CSS into styles.css.
- Moved Tailwind config into src/config/tailwind.config.js.
- Moved the original runtime/editor JavaScript into src/main.js.
- Kept the current UI and behaviour intact as much as possible.
- This is a preservation refactor, not a redesign.

Current structure:
- index.html
- styles.css
- src/main.js
- src/config/tailwind.config.js

Why this matters:
The original Gemini-generated file was too large to edit reliably. This first pass makes the app easier to review, commit, and split further.
