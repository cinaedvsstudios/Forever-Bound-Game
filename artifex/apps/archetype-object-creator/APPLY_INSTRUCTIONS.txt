Artifex Archetype Object Creator - Quickstart + Library Update

Copy these files into the repo, replacing the existing files:

1. artifex/apps/archetype-object-creator/index.html
2. artifex/apps/archetype-object-creator/v1/styles.css
3. artifex/apps/archetype-object-creator/v1/src/editor-ui.js

What this update adds:

- Centers the File / Edit / View / Templates / Library / Help menu row in the top bar on wide screens.
- Adds File -> Quick Start Wizard.
- Adds a Library menu with:
  - Open Archetype Object Library
  - Start From Existing Object
  - Browse Template Library
- Adds an Archetype Object Library dialog.
- Shows every object/person template as a card with a deliberately simple placeholder SVG image.
- Shows saved local archetypes in the library and lets them be loaded or copied as a new object.
- Adds a guided wizard flow:
  1. Choose Template or Existing Object
  2. Choose object type/template or saved existing object
  3. Choose what it can do: runtime flags, gameplay actions, portrait actions
  4. Make basic changes: ID, name, subtype, tags, sprite asset, portrait asset
  5. Finish or Finish + Save Local

Notes:

- This does not change templates.js. It uses the existing ROLE_TEMPLATES, PEOPLE_TEMPLATE_IDS, OBJECT_TEMPLATE_IDS, GAMEPLAY_ACTIONS, PORTRAIT_ACTIONS, and RUNTIME_FLAGS already in the module.
- Talk remains excluded from gameplay sprite actions. Dialogue/mouth movement stays in portrait actions.
- Existing object copying creates a new ID using the original ID plus a variant suffix, so the original local archetype is not overwritten.
