This is a non-FX copy of the Effect Editor shell architecture:
- top menu bar
- left editor panel
- canvas preview workspace
- bottom action panel
- local browser save
- JSON import/export
- templates
- validation
- preview snapshot

Effect-specific systems removed:
- particles
- emitter dynamics
- glow/colour-stop editing
- blend modes
- FX layers
- effect archetype presets

Important design rule:
Talk is not a gameplay sprite action. It belongs to the separate Dialogue Portrait actions.
