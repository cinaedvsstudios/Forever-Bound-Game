Artifex Shared Utility Preview

Sound Library + Create Synth Sound

The normal preview opens directly into the same shared Sound Library that future authoring apps will use. It includes the Create New Synth Sound route, but does not wire any caller app yet.

Use a disposable Blank Starter Project folder. The Sound Library reads assets/asset-index.json. Saving a generated synth writes assets/audio/sfx/synth_<slug>.json and registers one normal asset_sfx_... audio item.
