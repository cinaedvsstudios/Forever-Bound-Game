Artifex Quick Start

1. Insert a layer: Use Insert → Base Effect Layer to add a primitive engine, or Composite Effect to load a full preset.

2. Shape the look: Select a layer, then tune Visual Rendering for color, glow, size and blend mode.

3. Control motion: Motion Dynamics handles speed, angle, spread, gravity, drag and orbital swirl.

4. Save: Use File → Export JSON, or save a new reusable composition from the left panel.

Tip: Drag the blue splitters to resize the side panel and diagnostics area. Click the canvas to move the emitter manually.
