# Artifex Shimmer / Portal Distortion Engine Preview

V1.12 keeps the improved Portal Ring and specifically reworks Wormhole movement/shape.

## V1.12 changes

```text
Portal Ring is kept from V1.11 because the cloudy ring is now closer to target.
Wormhole Tunnel no longer relies mainly on concentric glowing ovals.
Wormhole now draws puffy spiral arms made from cloud blobs instead of thin circular strokes.
Wormhole low speed is now genuinely slow; the speed curve is squared so low values barely move.
Wormhole default speed is reduced.
Wormhole particles are stronger and follow the inward spiral path.
The giant uniform blue halo is reduced so the tunnel reads less like a flat glowing oval.
Cloud thickness now has a stronger effect on wormhole arm thickness.
```
