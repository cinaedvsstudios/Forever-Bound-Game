# Artifex Shimmer / Portal Distortion Engine Preview

V1.13 keeps the improved Portal Ring and fully changes the Wormhole visual language.

## V1.13 changes

```text
Portal Ring remains based on the better cloudy aperture from V1.11/V1.12.
Wormhole no longer defaults to a flat glowing eye/oval.
Wormhole default is closer to a round vortex tunnel.
Wormhole concentric ring dominance is removed.
Wormhole now uses a few distinct slow spiral arms, built from puffy cloud blobs.
Wormhole speed is even slower at low values by using a cubed speed curve.
Wormhole particles follow the inward spiral and are brighter by default.
The big uniform outer blue halo is reduced again.
Cloud thickness strongly affects the spiral arm mass.
```
