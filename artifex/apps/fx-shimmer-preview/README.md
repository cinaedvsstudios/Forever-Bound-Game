# Artifex Shimmer / Portal Distortion Engine Preview

V1.23 wires the requested portal and wormhole controls more directly into the renderer.

## V1.23 changes

```text
Portal line outline now has pulse strength and pulse speed.
Portal line outline thickness maximum is 3x stronger than before.
Portal line outline now has dedicated colour A and colour B controls.
Portal line outline colour mode supports solid, radial gradient, horizontal gradient, vertical gradient, diagonal gradient and image texture.
Portal line outline can now load a PNG/JPEG/WebP colour image.
Portal Ring now also renders the Orbit Clouds layer, so orbit-cloud controls work on portal effects too.
Wormhole Arm Radius is now dramatically connected to the arm wash, arm lobes and spiral arm strokes.
Wormhole Cloud Stagger now strongly offsets clouds across different radii instead of leaving them on one even orbit.
Wormhole Emission is strengthened with more particles, larger particles, stronger opacity and clearer trails.
Arm, cloud, particle and emission controls now use larger visual ranges so slider movement should be obvious.
```
