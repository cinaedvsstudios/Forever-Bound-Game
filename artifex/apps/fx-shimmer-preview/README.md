# Artifex Shimmer / Portal Distortion Engine Preview

V1.14 keeps the Portal Ring and rebuilds the Wormhole renderer around the reference direction: a rotating nebula/tunnel field, not a flat oval with a silly centre ring.

## V1.14 changes

```text
Portal Ring is left unchanged from the current good version.
Wormhole centre ring / eye detail is removed.
Wormhole now draws broad rotating spiral cloud arms instead of concentric oval bands.
The whole wormhole nebula rotates, not only the particles.
The wormhole is much lighter on per-frame drawing so it should not crawl as badly.
Default wormhole speed is restored to a visible but not frantic value.
The central vanishing point is now a subtle dark pull instead of a decorative ring.
Particles remain on the spiral path but are no longer the only moving element.
```
