# Phase 1 Testing Notes

This is the first manual playable shell.

## What should work

1. The game opens in the browser.
2. A fixed 16:9 game frame appears.
3. The title screen appears.
4. Start Game loads the forest scene.
5. The forest background uses `assets/scenes/ch00/ch00_q00_forest_route_bg1.png`.
6. Mel appears on the path.
7. Arrow Keys or WASD move Mel.
8. On-screen D-pad buttons also move Mel.
9. Mel changes facing direction using the four idle images.

## Required image files

```text
assets/scenes/ch00/ch00_q00_forest_route_bg1.png
assets/characters/mel/mel_idle_front.png
```

## Not included yet

No inventory, dialogue, enemies, Codice, Stone Marker, map, sound, crafting, or scrolling Travel Mode.
