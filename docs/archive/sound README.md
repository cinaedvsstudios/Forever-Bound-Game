# Sound Generator Preview

This folder is only the browser preview harness for the shared procedural sound generator.

The actual sound generator lives in:

```text
artifex/shared/sound-generator/
```

This preview imports:

```js
../../shared/sound-generator/sound-generator-window.js
```

Keep this folder if you want a simple page for testing the shared Create Synth Sound popup.

Active files:

```text
index.html
sound-generator-preview-host.js
README.md
```

This preview does not contain the synth engine, presets, runtime, save logic, or shared CSS.
