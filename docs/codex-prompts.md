# Codex Prompts

## Phase 1 Prompt

We are in Phase 1 of the Forever Bound companion game.

Create a Vite + TypeScript browser game project.

Technical rules:
- Use HTML Canvas for the gameplay area.
- Use a fixed 16:9 game frame.
- Preserve aspect ratio with black bars instead of stretching.
- Create a title screen with Start Game, Continue, Options, and Credits buttons.
- Start Game should load a placeholder gameplay scene.
- Add a placeholder HUD at the top with hearts, silver, active item, and status box.
- Add NES-style on-screen controls at the bottom: D-pad, A, B, X, Y, Select, Start.
- The buttons do not need full gameplay functions yet, but they should visually exist.
- Use placeholder shapes or simple temporary assets.
- Keep the code modular.
- Do not add inventory, combat, enemies, dialogue, map, crafting, Codice, or quests yet.

Done means:
- npm install works.
- npm run dev works.
- the title screen opens in the browser.
- clicking Start Game shows the placeholder gameplay scene with HUD and controls.
